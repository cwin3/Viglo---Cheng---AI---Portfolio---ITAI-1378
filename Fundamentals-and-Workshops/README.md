# Computer Vision Fundamentals & Workshops

## Overview
This folder contains a series of laboratory and workshop notebooks that
introduce core computer vision and deep learning concepts.

## Notebooks Included
- L02: Introduction to computer vision concepts
- L04: Workshop exercises and experimentation
- L06: Neural network fundamentals
- L08: Additional model training and evaluation

## Key Learning Outcomes
- Understanding image data representation
- Applying neural networks to visual tasks
- Interpreting training results and model behavior

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
